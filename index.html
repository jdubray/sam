The SAM Pattern

SAM is a new reactive/functional pattern which simplifies Front-End architectures by clearly separating the business logic from the view, and in particular strictly decoupling back-end APIs from the Front-End.

SAM is unapologetically driven by simplicity and challenges the complexity of Frameworks like Google's Angular and Facebook's React+Redux+GraphQL+Relay.

SAM is also challenging architecture patterns like BFF or the Vertical Slice Pattern which suggest creating dedicated APIs for each view, per platform, app, versions of an app...
